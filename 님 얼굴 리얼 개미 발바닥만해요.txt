상단 중앙 : 표지 제목(완성)
상단 우측 : 오늘 날씨(기상청 오늘 날씨 DB 백엔드 포함)(미완: 코딩 미숙)
전체 화면 배경 : black and white polka dot pattern(완성)(그림 탐색중)
좌측 : 애니메이션 줘서 리스트 출력(미완: 조언 구할 예정)
우측 : 리모컨 - input type="checkbox", textbox, css cursor, js(미완: 시간 남으면 하고 안남으면 미실시)
중앙 : 신문에 실린 기사처럼 각 문단을 나눠서 표현, 각 문단은 줄을 표시해서 나타낸다.(완성)
문단 : 각 게시판의 맨 윗줄(DB 백엔드)를 기반으로 그림 및 영상 출력, 해당 글을 표지에 출력(미완: 반 정도 완성)
